Matlab demo code for "K-means Hashing: an Affinity-Preserving Quantization Method

for Learning Binary Compact Codes" (CVPR 2013)

by Kaiming He (kahe@microsoft.com), Mar. 2013

If you use/adapt our code, please appropriately cite our CVPR 2013 paper.

This code is for academic purpose only. Not for commercial/industrial activities.

***************************************************************************************
***************************************************************************************

Usage:

Training:

1. Edit run.m to set parameters and datasets to use
2. Run run.m in Matlab

Searching:

1. Edit search.sh to set parameters and the dataset to use
2. Run `./kmh_search.sh`

Acknowledgement:

Most of the codes in this directory was obtained from https://github.com/jiaxiang-wu/libhash.
